## 客户管理后端接口

### 接口1：多条件分页查询客户列表

- 请求方法：GET
- url : `/customer/page`
- 参数: 

| 参数名  | 说明                     |
| ------- | ------------------------ |
| code    | 客户编号，字符串，默认空 |
| name    | 客户名，字符串，默认空   |
| status  | 客户状态，默认空         |
| pageNum | 页码，默认1              |
| size    | 每页大小，默认10         |

响应结果：

```json
{
  "code": 200,
  "msg": "操作成功",
  "data": {
      "total": 12,
    "list": [
      {
        "id": 1,
        "code": "CUST001",
        "name": "华为技术有限公司",
        "address": "深圳市龙岗区华为总部",
        "contactPerson": "张经理",
        "phone": "0755-12345678",
        "fax": "0755-87654321",
        "email": "zhang@huawei.com",
        "description": "重要客户",
        "status": 1,
        "createTime": "2025-09-06T14:56:48",
        "updateTime": "2025-09-06T14:56:48"
      },
      .......
    ]
  }
}
```

### 接口3：新增客户

- 请求方法：POST
- url ： `/customer/add`
- 参数

参数以JSON格式放在请求体中提交

参数示例：

```json
{
        "code": "CUST001",
        "name": "华为技术有限公司",
        "address": "深圳市龙岗区华为总部",
        "contactPerson": "张经理",
        "phone": "0755-12345678",
        "fax": "0755-87654321",
        "email": "zhang@huawei.com",
        "description": "重要客户",
        "status": 1
      }
```

- 响应结果

成功：

```json
{
    "code":200,
    "msg":"操作成功",
   	"data":{
        // 这里为保存成功的客户信息
    }
}
```

失败：

```json
{
    "code":500,
    "msg":"操作失败",
   	"data":null
}
```
