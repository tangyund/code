##  供应商表 (supplier)

| 字段名         | 数据类型   | 长度 | 允许空 | 默认值            | 主键 | 唯一键 | 索引 | 注释                                  |
| -------------- | ---------- | ---- | ------ | ----------------- | ---- | ------ | ---- | ------------------------------------- |
| id             | bigint     |      | 否     |                   | 是   |        |      | 供应商编号                            |
| code           | varchar    | 50   | 否     |                   |      | 是     |      | 供应商编码                            |
| name           | varchar    | 100  | 否     |                   |      |        |      | 供应商名称                            |
| address        | varchar    | 200  | 是     | NULL              |      |        |      | 供应商地址                            |
| contact_person | varchar    | 50   | 是     | NULL              |      |        |      | 供应商联系人                          |
| phone          | varchar    | 20   | 是     | NULL              |      |        |      | 供应商电话                            |
| fax            | varchar    | 20   | 是     | NULL              |      |        |      | 供应商传真                            |
| email          | varchar    | 100  | 是     | NULL              |      |        |      | 供应商邮箱                            |
| description    | text       |      | 是     | NULL              |      |        |      | 供应商描述                            |
| status         | tinyint(1) |      | 否     | 1                 |      |        |      | 是否启用(0:禁用,1:启用)               |
| create_time    | datetime   |      | 是     | CURRENT_TIMESTAMP |      |        |      | 创建时间                              |
| update_time    | datetime   |      | 是     | CURRENT_TIMESTAMP |      |        |      | 更新时间(ON UPDATE CURRENT_TIMESTAMP) |
| delete_flag    | int        |      | 否     | 0                 |      |        |      | 删除标记                              |



## 供应商管理相关接口说明

### 接口1：多条件分页查询供应商列表

- 请求方法：GET
- url : `/supplier/page`
- 参数: 

| 参数名  | 说明                       |
| ------- | -------------------------- |
| code    | 供应商编号，字符串，默认空 |
| name    | 供应商名，字符串，默认空   |
| status  | 供应商状态，默认空         |
| pageNum | 页码，默认1                |
| size    | 每页大小，默认10           |

### 接口3：新增供应商

- 请求方法：POST
- url ： `/supplier/add`
- 参数

参数以JSON格式放在请求体中提交

参数示例：

```json
{
    "code": "SUPP003",
    "name": "宁德时代新能源科技股份有限公司",
    "address": "福建省宁德市宁德时代总部",
    "contactPerson": "王经理",
    "phone": "0593-55555555",
    "fax": "0593-66666666",
    "email": "wang@catl.com",
    "description": "电池供应商",
    "status": 1,
  }
```

- 响应结果

成功：

```json
{
    "code":200,
    "msg":"操作成功",
   	"data":{
        // 这里为保存成功的客户信息
    }
}
```

失败：

```json
{
    "code":500,
    "msg":"操作失败",
   	"data":null
}
```

### 接口5：根据id删除供应商信息

- 请求方法： DELETE
- url: `/supplier/delete/{id}`
- 请求参数

参数在清路径中｛id｝，为供应商id

- 响应结果

成功：

```json

    "code":200,
    "msg":"操作成功",
   	"data":null
}
```

失败：

```json
{
    "code":500,
    "msg":"操作失败",
   	"data":null
}
```
