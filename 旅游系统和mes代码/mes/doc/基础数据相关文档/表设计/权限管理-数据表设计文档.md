
## 表设计文档

### 1. 用户表（sys_user）
存储系统用户的基本信息

| 字段名 | 数据类型 | 长度 | 主键 | 允许为空 | 说明 |
|--------|----------|------|------|----------|------|
| id | BIGINT | - | 是 | 否 | 用户ID |
| username | VARCHAR | 50 | 否 | 否 | 用户名，唯一 |
| password | VARCHAR | 100 | 否 | 否 | 加密后的密码 |
| full_name | VARCHAR | 100 | 否 | 是 | 用户姓名 |
| email | VARCHAR | 100 | 否 | 是 | 电子邮箱 |
| phone | VARCHAR | 20 | 否 | 是 | 手机号码 |
| status | TINYINT | - | 否 | 否 | 状态：0-禁用，1-启用 |
| avatar | VARCHAR | 255 | 否 | 是 | 头像URL |
| last_login_time | DATETIME | - | 否 | 是 | 最后登录时间 |
| created_by | BIGINT | - | 否 | 是 | 创建人ID |
| created_time | DATETIME | - | 否 | 否 | 创建时间 |
| updated_by | BIGINT | - | 否 | 是 | 更新人ID |
| updated_time | DATETIME | - | 否 | 是 | 更新时间 |
| remark | VARCHAR | 500 | 否 | 是 | 备注 |
| delete_flag | int |  | 否 | 0 |  |

### 3. 权限表（sys_permission）
存储系统权限信息

| 字段名 | 数据类型 | 长度 | 主键 | 允许为空 | 说明 |
|--------|----------|------|------|----------|------|
| id | BIGINT | - | 是 | 否 | 权限ID |
| parent_id | BIGINT | - | 否 | 是 | 父权限ID，顶级权限为0 |
| permission_name | VARCHAR | 100 | 否 | 否 | 权限名称 |
| permission_code | VARCHAR | 100 | 否 | 否 | 权限编码，唯一 |
| permission_type | TINYINT | - | 否 | 否 | 权限类型：1-菜单，2-按钮，3-接口 |
| path | VARCHAR | 255 | 否 | 是 | 资源路径 |
| component | VARCHAR | 255 | 否 | 是 | 组件路径 |
| icon | VARCHAR | 100 | 否 | 是 | 图标 |
| sort | INT | - | 否 | 是 | 排序 |
| status | TINYINT | - | 否 | 否 | 状态：0-禁用，1-启用 |
| created_by | BIGINT | - | 否 | 是 | 创建人ID |
| created_time | DATETIME | - | 否 | 否 | 创建时间 |
| updated_by | BIGINT | - | 否 | 是 | 更新人ID |
| updated_time | DATETIME | - | 否 | 是 | 更新时间 |
| remark | VARCHAR | 500 | 否 | 是 | 备注 |
| delete_flag | int |  | 否 | 0 |  |

### 6. 登录日志表（sys_login_log）
记录用户登录日志

| 字段名 | 数据类型 | 长度 | 主键 | 允许为空 | 说明 |
|--------|----------|------|------|----------|------|
| id | BIGINT | - | 是 | 否 | 日志ID |
| user_id | BIGINT | - | 否 | 是 | 用户ID |
| username | VARCHAR | 50 | 否 | 否 | 用户名 |
| ip_address | VARCHAR | 50 | 否 | 是 | 登录IP地址 |
| login_location | VARCHAR | 100 | 否 | 是 | 登录地点 |
| browser | VARCHAR | 50 | 否 | 是 | 浏览器 |
| os | VARCHAR | 50 | 否 | 是 | 操作系统 |
| login_time | DATETIME | - | 否 | 否 | 登录时间 |
| login_status | TINYINT | - | 否 | 否 | 登录状态：0-失败，1-成功 |
| msg | VARCHAR | 255 | 否 | 是 | 登录信息 |
