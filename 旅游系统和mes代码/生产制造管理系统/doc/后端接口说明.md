### 工艺流程管理后端接口

#### 接口1：多条件查询工艺

- 请求方法 ：GET
- url ： `/workflow/page`
- 参数：

| 参数         | 说明                           |
| ------------ | ------------------------------ |
| page         | 当前页码                       |
| size         | 每页大小                       |
| workflowCode | 工艺流程编码，字符串，可以为空 |
| workflowName | 工艺流程名称，字符串，可以为空 |
| status       | 状态，整形，可以为空           |

- 响应结果

```json
{
    "code": 200,
    "msg": "操作成功",
    "data": {
        "total": 15,
        "list": [
            {
                "id": 1,
                "workflowCode": "WF001",
                "workflowName": "手机组装工艺流程",
                "status": 1,
                "workflowDescription": "智能手机整机组装全流程",

                "createBy": "admin",
                "updateTime": "2025-09-10 16:53:50",
                "updateBy": "admin",
                "deleteFlag": 0
            },
            .......
        ]
    }
}
```
